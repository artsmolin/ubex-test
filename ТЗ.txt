Тестовое задание 
для backend разработчика
Условия:
Для реализации тестового задания можно использовать любые доступные библиотеки. 
Решение должно быть самодостаточным и содержать всё необходимое для запуска, в том числе инструкции по установке и запуску. 
Покрытие кода решения тестами будет плюсом.
Решение должно быть выложено в публичный репозиторий на github.com или bitbucket.org.

Задача:
Требуется написать расширение, реализующее автоматическое REST API для всех моделей в Django проекте.

Возможности:
Система должна распознавать все существующие модели, подключённых в проект приложений и публиковать REST интерфейс для внешнего взаимодействия 
API должно позволять:
запрашивать объекты по нескольким полям (логика - AND) с возможностью сортировки (ORDER BY), ограничения кол-ва (LIMIT) (метод GET)
создавать новые объекты (метод POST)
удалять объекты по первичному ключу (метод DELETE)
обновлять объекты по первичному ключу (метод PUT)
Данные клиенту должны возвращаться в формате JSON
